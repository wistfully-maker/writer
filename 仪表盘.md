# 📊 故事花园仪表盘

## 章节总览
```dataview
TABLE 
  word_count AS "字数", 
  score AS "读者评分", 
  status AS "状态"
FROM "wiki/章节"
SORT number ASC
```

## 全书进度统计
```dataview
TABLE WITHOUT ID
  length(rows) AS "章节数",
  sum(rows.word_count) AS "总字数",
  round(avg(rows.score), 1) AS "平均评分"
FROM "wiki/章节"
FLATTEN word_count, score
GROUP BY true
```

## 未回收伏笔
```dataview
TABLE 
  planted_in AS "埋设场景",
  planned_payoff AS "计划回收章",
  status AS "状态"
FROM "wiki/结构"
WHERE type = "foreshadowing" AND status != "paid_off"
SORT planned_payoff ASC
```

## 活跃人物状态
```dataview
TABLE 
  arc_status AS "弧光进度",
  last_relevant_chapter AS "最后活跃章"
FROM "wiki/人物"
WHERE status = "active"
SORT arc_status ASC
```
