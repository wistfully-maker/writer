# Skill：哲思熔铸法

## 技能描述
将道教、佛教等传统文化哲学思想以及作者的人生感悟，无缝融入角色对话、情节设计和世界观规则中，使其成为推动叙事的内在逻辑，而非说教贴片。

## 适用条件
- 设计世界观底层规则时（如内景、性命双修）。
- 塑造高境界角色（如老天师、宝儿姐）的价值观和行事逻辑时。
- 需要为情节冲突提供超越简单善恶的深层理由时。
- 想要传达某种人生感悟（如"顺应天性"、"承负"）时。

## 执行指令（给 Architect/Writer 的指导）
1. **提炼哲学核心**：从道家、佛家等思想中提炼一个具体概念（如《一人之下》中的"真"、"诚"、"无所得"），使其成为贯穿故事的主题之一，写入主题卡。
2. **规则化**：将抽象哲理转化为具体的世界观规则或力量体系。例如："修行的关键不是变强，而是回归先天一炁的'真'，任何杂念都会成为心魔。"
3. **人物化身**：让不同角色成为不同哲学观点的载体。他们的冲突不仅是利益之争，更是理念之争（如全性"随心所欲"与天师府"持戒守心"的碰撞）。
4. **台词生活化**：让人物在关键对话中用大白话道出深刻哲理，避免长篇大论。通常是在极端情境下，一句朴素的话点破天机（如"你太想进步了，所以停在这里"）。
5. **情节验证**：设计一个情节，让主角通过践行或违背某种哲学理念而导致结果的天壤之别，从而自然引出人生感悟，而不是直接告诉读者。

## 评判检验点（给 Reader 的检查项）
- [ ] 世界观规则是否体现了某个可辨识的哲学思想？
- [ ] 角色的重大选择是否由其内在的价值观（而非剧情需要）驱动？
- [ ] 哲理表达是否通过具体情节和动作呈现，而非通过讲道或内心旁白？
- [ ] 不同角色的理念冲突是否增加了故事的张力？
- [ ] 是否有生硬"背书"或掉书袋的违和感？
