# Skill：情绪循环设计

## 技能描述
通过"憋屈→局部释放→更大的憋屈→全面爆发"的循环公式，确保高潮场景的情绪兑现力，让读者获得酣畅淋漓的满足感。

## 适用条件
- 高潮/反杀/大节点场景。
- 需要检查压抑积累是否充分时。
- 规划卷弧的情绪节奏时。

## 执行指令（给 Architect/Writer 的指导）
1. **循环公式**：憋屈 → 局部释放 → 更大的憋屈 → 全面爆发。
2. **憋屈来源必须"结构性"**：压迫感来自体制、信息不对等、道德困境、资源匮乏，而非主角突然降智或无理取闹。
3. **局部释放**：在全面爆发前，必须安排 1-2 个"小型的翻身"或"暂时的胜利"，让读者喘气并建立"主角确实有能力翻盘"的信念。
4. **全面爆发**：高潮场景必须兑现之前累积的所有情绪，不留下"没爽到位"的遗憾。反馈要分层（配角反应、环境变化、后续影响）来渲染重要性。

执行边界：
- **严禁**用"主角突然变蠢"来制造憋屈。憋屈必须让读者觉得"没办法，换成我也得忍着"。
- **严禁**全面爆发后主角陷入更深的无意义麻烦，那叫"添堵"，不叫循环。
- **不适用于**：纯日常或感情线，不必强行憋屈，可切换为"温馨→小波澜→温馨"的轻循环。

## 评判检验点（给 Reader 的检查项）
- [ ] 压抑阶段是否让我感到"胸口发闷"或"愤愤不平"（而非"想弃书"）？
- [ ] 局部释放是否真实地改善了一点局势，而不只是主角的错觉？
- [ ] 全面爆发时，是否有至少两个维度（如战斗力、智力、道德）共同作用？
- [ ] 反派失败时，周围群众的反应是否有阶段变化（怀疑→凝固→疯狂议论）？

## 反例警告
- 最低劣形式：反派无故嘲讽 → 主角爆发 → 反派跪地求饶，全程没有任何智力博弈。
- 憋屈篇幅过长（超过 10 章无任何释放），读者会因"太虐"而弃书。
