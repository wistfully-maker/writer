# Skill：命名暗线法

## 技能描述
赋予角色姓名以"反向张力"或"命运暗示"，使名字成为塑造人物、埋伏笔的微型叙事工具。

## 适用条件
- 创建任何重要角色（主角、配角、反派）时。
- 需要让人物在出场瞬间就携带"隐藏信息"时。
- 想通过名字本身制造悬念或反讽时。

## 执行指令（给 Architect/Writer 的指导）
1. **创建角色时**，不要使用随机生成的名字。先明确角色的核心矛盾：他的外在表现与内在真实之间的最大冲突是什么？
2. **选择名字中的一个字**，使其字面含义与角色的真实性格/命运构成鲜明对比（如吕慈不慈，王蔼不蔼），或形成命运的讽刺（如肖自在从未自在）。
3. 如果角色有特殊能力或使命，名字可暗含其能力本质或悲剧宿命（如"楚岚"暗藏风云变幻）。
4. **在首次出场时**，通过一个小细节让读者隐约感觉到"名不副实"的违和感，但不点破。后续情节中逐渐揭示名字的真正含义。
5. **记录到人物卡**：在人物卡的"身份表面层"或新增的"命名暗线"字段中，写下这个命名的内在矛盾，并计划它在何时被揭示或引爆。

## 评判检验点（给 Reader 的检查项）
- [ ] 重要角色的名字是否存在字面含义与真实性格/命运的明确反差？
- [ ] 名字是否在情节中起到了暗示或伏笔作用？
- [ ] 是否避免了"贴标签"式的浅白命名（如坏人直接叫"恶"）？
- [ ] 名字的反讽是否最终得到了情节上的呼应（肖自在的"不自在"是否有事件体现）？
