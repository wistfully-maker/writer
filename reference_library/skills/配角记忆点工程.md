# Skill：配角记忆点工程

## 技能描述
通过"三位一体"（标志性动作/口头禅 + 价值立场 + 与主角关系定位）确保每个重要配角都能被读者瞬间识别和记住。

## 适用条件
- 引入新配角或配角重要出场时。
- 检查已有配角是否沦为背景板时。
- 规划配角弧光时。

## 执行指令（给 Architect/Writer 的指导）
1. **三位一体**：一个标志性动作/口头禅 + 一个鲜明的价值立场 + 一个与主角关系的独特定位。
2. **一致性**：此配角的任何出场，都必须展示至少其中两项，确保读者能瞬间识别。
3. **价值立场**：记忆点不是外貌标签，而是该角色在故事道德光谱上的位置（如"绝对的利己主义者，但从不掩饰"）。这个立场必须在其行为中反复被印证。
4. **成长性**：重要的配角应该有微小的价值立场位移，但不允许突然颠覆（除非有足够的铺垫）。

执行边界：
- **严禁**将记忆点简化为"一个怪癖"。反派不能只会"桀桀桀"，女配不能只会"脸红"。
- **严禁**让配角的言行与其价值立场矛盾。一个"冷血杀手"不会因主角几句话就感动流泪。
- **不适用于**：纯粹的功能性龙套（如店小二、路人），不必强加记忆点，用一次就消失的角色就让他普通。

## 评判检验点（给 Reader 的检查项）
- [ ] 过去 10 章出现的配角中，我能清晰回忆出至少 3 个的特征吗？
- [ ] 这个配角的标志性动作/台词是否与其价值立场一致，而非突兀？
- [ ] 配角的行为有没有因为"剧情需要"而突然改变？如果是，标记为人物崩塌。
- [ ] 主角对待这个配角的方式，是否与该配角的定位稳定匹配？

## 反例警告
- 最低劣形式：所有手下都"倒吸一口凉气"，所有女友都"眼中闪过一丝狡黠"。
- 配角在后期被完全遗忘或沦为背景板，说明记忆点工程失败。
