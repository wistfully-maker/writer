# Skill：对话潜台词密度

## 技能描述
确保关键对话中的每句对白都承载说话人的目的、态度或预判，让人物对话成为权力博弈而非信息传递。

## 适用条件
- 含重要对话的场景（对峙、谈判、试探、告白）。
- 检查是否存在"工具人对白"时。
- 需要通过对话揭示人物关系时。

## 执行指令（给 Architect/Writer 的指导）
1. **潜台词定义**：每一句对白，除了字面信息之外，必须至少承载以下三项之一：
   - **说话人的目的**（为什么要说这句话？想得到什么反应？）
   - **说话人的态度**（对听者的真实看法是什么？）
   - **说话人的预判**（认为听者会怎么理解这句话？）
2. **密度标准**：关键场景（谈判、对峙、试探、情感告白）中，潜台词对白占比须超过 50%。日常场景不做强制要求，但至少核心人物的对白应体现其价值立场。
3. **信息传递的伪装**：如果某个信息必须通过对话告知读者，必须包装在人物的目的和态度之下，不能让人物变成"作者的工具人"。
4. **沉默也是台词**：关键对话中，人物的回避、停顿、转移话题、答非所问，都是潜台词的一部分。

执行边界：
- **严禁**将所有对话都写得高深莫测或阴阳怪气。日常闲聊、问路、点菜等场景不需要潜台词。
- **严禁**让人物在对话中完整复述读者已经知道的事情。如果必须提醒读者，用一句话概括而非重复原文。
- **不适用于**：纯粹的背景介绍型对话（如说书人、史官、讲师），可以直白，但需控制长度。

## 评判检验点（给 Reader 的检查项）
- [ ] 本章关键场景的对话中，我是否能感觉到每个人物的真实意图（包括他们想隐藏的）？
- [ ] 有没有哪句对白让我觉得"这话是作者为了告诉读者信息而硬塞的"？
- [ ] 对话双方的回答是真正在互动，还是各说各话？
- [ ] 有没有一段对白超过 5 轮以上，但除了信息交换之外没有任何人物态度的变化？

## 反例警告
- 最低劣形式 A：两个角色互相解释他们都已经知道的事情，只为让读者听到。
- 最低劣形式 B：所有角色的对话风格都一样，你分不清谁在说话。
- 最低劣形式 C：反派在打斗或失败时突然开始大段独白解释自己的动机。
