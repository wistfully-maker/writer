# Skill：东晋式"暧昧与后宫塑造"手法

## 技能描述
通过克制的暧昧张力、每个女性角色独特的情感表达方式、以及"主角不是在撩而是在认真对待每一个人"的核心逻辑，让后宫群像不沦为工具人集合，而是每个读者都能找到"自己站的CP"。

## 适用条件
- 场景涉及核心女性角色首次出场。
- 场景涉及情感互动升级（从陌生到心动、从暧昧到确认）。
- 场景涉及后宫群像调度（多女性角色同框）。
- 人类明确要求增强情感线。

## 执行指令（给 Architect/Writer 的指导）
1. **首次出场的"三秒定型"**：女性角色首次出场时，必须在三个自然段内完成一个"只属于她"的标志性瞬间——可以是一个动作、一句话、一个表情。这个瞬间必须同时展示外貌和性格，而非先描述外貌再展示性格。
2. **暧昧的"距离感管理"**：暧昧的张力来自"差一点"。眼神交汇但先移开、话到嘴边又咽回去、手指碰到又缩回。让读者感到"就差那么一点点"，比直接写"他们接吻"有效十倍。
3. **每条情感线的独特频率**：每个女性角色与主角的互动模式必须不同。一个是斗嘴型、一个是默契型、一个是保护型、一个是被保护型。频率相同会让读者混淆。
4. **"认真对待"的底线**：主角对每一个女性角色的关心必须是真实的、有逻辑的，而非"为了不让她伤心所以不拒绝"。读者能分辨"温柔"和"渣"的区别。
5. **群像同框时的"舞台调度"**：多女性角色同时出现时，必须有明确的"主舞台"和"侧写"。不要试图给每个人均等的篇幅，而是让一两个人成为焦点，其他人的反应通过一两个细节带过。

## 评判检验点（给 Reader 的检查项）
- [ ] 女性角色首次出场时，我是否在三个段落内就记住了她"不一样"的地方？
- [ ] 暧昧场景是否让我感到"就差一点点"的心痒，而非"作者在故意撩我"的尴尬？
- [ ] 我能否用一句话区分任意两个女性角色的情感表达方式？
- [ ] 主角对每个女性角色的态度是否有清晰的、可解释的理由？
- [ ] 群像同框时，我是否能清晰分辨谁是"主舞台"谁是"侧写"？

## 反例警告
- 最低劣形式：所有女性角色都"脸红"、"跺脚"、"哼了一声"，你分不清谁是谁。
- 主角对所有女性角色都说"你是我最重要的人"，那每个人都不是最重要的。
- 暧昧场景直接跳到"他们发生了关系"，中间没有任何张力积累。
