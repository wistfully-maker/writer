# Skill：江南式"衰小孩"共鸣与细腻文笔

## 技能描述
通过精准的心理独白、少年式的自卑与自嘲、以及细腻到近乎疼痛的感官描写，让读者对"不完美的普通人主角"产生深层共鸣。核心是让读者在主角身上看到曾经那个"觉得自己不够好"的自己。

## 适用条件
- 场景类型为第一人称或第三人称限制视角，且需渲染孤独感、自卑感、少年感、怀旧感。
- 人类明确要求"更有江南的味道"。
- 主角处于被忽视、被嘲笑、自我怀疑的阶段。

## 执行指令（给 Architect/Writer 的指导）
1. **心理独白的质感**：用短句和具体细节代替抽象情绪。不写"他很难过"，写"他把书包带子攥得很紧，指节发白，但他没有回头"。
2. **自卑的表达方式**：主角对自己的评价永远比别人对他的评价低一档。他会在心里默默比较，然后得出一个自嘲的结论。
3. **感官的放大**：在情绪高涨的时刻，一个微小的感官细节（气味、声音、触感）被放大到不合理的程度，用来暗示主角的情感浓度。
4. **对比与反差**：越是内心翻涌，外在表现越平静。越是在乎，嘴上越轻描淡写。这种反差是制造心疼感的核心手法。
5. **少年式的比喻**：比喻来自校园生活、流行文化、日常琐碎，而不是文学经典。一个高中生不会用"如同西西弗斯的巨石"来形容压力，他会说"像考试时发现最后一页没印到"。

## 评判检验点（给 Reader 的检查项）
- [ ] 主角的内心独白是否让我想起自己少年时的某个瞬间？
- [ ] 有没有用具体感官细节代替抽象情绪？（"他的手在发抖"好过"他感到恐惧"）
- [ ] 主角的自嘲是否真实而非刻意卖惨？
- [ ] 情绪高潮时，是否有至少一个被放大的感官细节？
- [ ] 比喻是否来自角色的生活经验而非作者的文学储备？

## 反例警告
- 最低劣形式：主角不断内心独白"我好惨我好没用"，但没有任何具体行为佐证。
- 为了"细腻"而堆砌形容词，一句话里超过三个形容词就会失去力量。
