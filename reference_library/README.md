# Reference Library — 参考素材库

本目录存储小说创作系统的所有参考素材和技能模块。

## 目录结构

```
reference_library/
├── skills/                    # 所有已完成的 Skill 卡（Agent 直接调用）
│   ├── 悬念分层管理.md         # 基础技能：三层悬念结构
│   ├── 情绪循环设计.md         # 基础技能：憋屈→释放循环
│   ├── 配角记忆点工程.md       # 基础技能：三位一体记忆点
│   ├── 世界观碎片化释放.md     # 基础技能：行动中展示设定
│   ├── 对话潜台词密度.md       # 基础技能：对白承载目的/态度
│   ├── 反套路设计.md           # 基础技能：伏笔前置的反转
│   ├── 江南式衰小孩共鸣.md     # 进阶技能：孤独感/少年感文笔
│   ├── 东晋式暧昧与后宫塑造.md # 进阶技能：暧昧张力与群像调度
│   ├── 命名暗线法.md           # 进阶技能：名字与命运的反差
│   └── 哲思熔铸法.md           # 进阶技能：哲学思想融入设定
│
├── materials/                 # 未加工的原始素材（人类参考，Agent 不调用）
│   ├── 网文片段/              # 网文名场面、经典桥段
│   └── 名著片段/              # 文学名著节选
│
├── templates/                 # 新建 Skill 卡时的格式模板
│   └── skill_card_template.md
│
└── README.md                  # 本文件
```

## 技能分级

### 基础技能（常驻）
每次创作自动内化，无需额外触发。Writer 和 Reader 的 prompt 中已嵌入核心原则。

| skill_id | 名称 | 核心原则（一行） |
|----------|------|-----------------|
| `suspense_layering` | 悬念分层管理 | 每章结尾必须有一个可辨识的表层悬念 |
| `dialogue_subtext` | 对话潜台词密度 | 关键对话中，人物说的和想的永远有距离 |
| `sidechar_memory` | 配角记忆点工程 | 有名字的配角出场，必须有一个瞬间让读者记住 |
| `worldbuilding_fragments` | 世界观碎片化释放 | 绝不通过旁白灌输设定，让规则在行动中展现 |
| `emotion_cycle` | 情绪循环设计 | 高潮前必须积累压抑，爆发时必须有周围反馈 |
| `anti_cliche` | 反套路设计 | 任何反转必须有前文伏笔支撑 |

### 进阶技能（触发式）
仅在特定条件满足时激活，由架构师读取对应 skill 卡并融入场景指令。

| skill_id | 名称 | 触发条件 |
|----------|------|---------|
| `jiangnan_resonance` | 江南式衰小孩共鸣 | 需渲染孤独/自卑/少年感/怀旧感 |
| `dongjin_harem` | 东晋式暧昧与后宫塑造 | 女性角色首次出场/情感升级/群像调度 |
| `naming_subtext` | 命名暗线法 | 创建新的重要角色 |
| `philosophy_integration` | 哲思熔铸法 | 设计世界观规则/高境界角色/理念冲突 |

## 新 Skill 入库流程

1. 将原始文本片段放入 `materials/` 对应子目录
2. 参考 `templates/skill_card_template.md`，生成 Skill 初稿
3. 人类审核、修改、确认后，将成品放入 `skills/`
4. 确保 frontmatter 中 `verified: true`、`skill_id` 唯一
5. 更新本 README 的技能列表
